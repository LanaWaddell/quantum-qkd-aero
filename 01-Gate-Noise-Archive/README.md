# Aero Telemetry Sandbox

This is a secure R&D environment for Python-based aerospace telemetry analysis.

## Project Structure

- `simulations/`: Subfolder for aerospace simulation scripts and data.
- `README.md`: Project overview and documentation.

## Setup

- **Python Interpreter**: Homebrew Python 3.13.12 (via virtual environment)
- **Environment**: Managed using the `venv/` directory.
- **Dependencies**: `qiskit`, `qiskit-aer`

## Usage

To run the quantum telemetry simulation:
```bash
./venv/bin/python3.13 quantum_test.py
```

- `quantum_test.py`: Bell State circuit simulating synchronized satellite sensors.
